# Технический стек

## Язык программирования

### Python 3.x
Основной язык разработки проекта. Рекомендуемая версия: Python 3.10+

Преимущества:
* Богатая экосистема библиотек для работы с геометрией и научными вычислениями
* Хорошая производительность для численных расчетов
* Кроссплатформенность
* Активное сообщество разработчиков

## GUI Framework

### PySide6 / PyQt6
Qt-фреймворк для создания пользовательского интерфейса.

Выбор между PySide6 и PyQt6:
* **PySide6** - официальная версия от Qt Company (рекомендуется)
* **PyQt6** - альтернативная реализация

Основные компоненты:
* QMainWindow для главного окна
* QWidget для кастомных компонентов
* QGraphicsView для 2D-визуализации
* Интеграция с 3D-viewport

## 3D-визуализация

### PyVista
Высокоуровневый Python-интерфейс для VTK, упрощающий работу с 3D-данными.

Возможности:
* Встроенные инструменты для работы с mesh
* Интерактивные виджеты
* Интеграция с Qt

### VTK (The Visualization Toolkit)
Мощная библиотека для 3D-визуализации и обработки данных.

Использование:
* Рендеринг 3D-сцен
* Интерактивное взаимодействие
* Обработка больших объемов данных

### OpenGL (частично)
Для специфических задач, требующих низкоуровневого доступа к графическому конвейеру.

## Работа с геометрией

### B-Rep (Boundary Representation)

#### pythonocc (Python OCC)
Python-обертка над OpenCASCADE Technology для работы с CAD-геометрией.

Функциональность:
* Импорт/экспорт STEP
* Операции над B-Rep
* Геометрические вычисления

#### cadquery
Декларативный API для создания CAD-моделей.

Использование:
* Программное создание моделей
* Параметрическое моделирование
* Интеграция с pythonocc

### Mesh

#### trimesh
Библиотека для работы с треугольными сетками.

Возможности:
* Загрузка/сохранение mesh-форматов (STL, OBJ, PLY)
* Операции над mesh (упрощение, ремонт, булевы операции)
* Анализ геометрии
* Визуализация

#### Open3D
Библиотека для работы с 3D-данными, включая point cloud и mesh.

Использование:
* Обработка point cloud
* Алгоритмы реконструкции поверхности
* Визуализация
* ICP (Iterative Closest Point) для выравнивания

## Численные расчеты

### NumPy
Фундаментальная библиотека для научных вычислений.

Применение:
* Работа с массивами данных
* Линейная алгебра
* Математические операции над геометрией

### SciPy
Библиотека научных вычислений, построенная на NumPy.

Использование:
* Оптимизация
* Интерполяция
* Обработка сигналов
* Специальные функции

## Machine Learning

### PyTorch
Фреймворк для машинного обучения.

Использование:
* Локальный inference (не требуется GPU, но поддерживается)
* Предобученные модели для анализа геометрии
* Возможность дообучения (опционально)

Модели:
* Классификация дефектов
* Предсказание оптимальной ориентации
* Обнаружение проблемных зон

## FEM-модуль (опционально)

Для упрощенного прочностного анализа печатных деталей могут использоваться:
* FEniCS (для более сложных задач)
* Собственная упрощенная реализация на NumPy/SciPy
