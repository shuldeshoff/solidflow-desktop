# FAQ - Часто задаваемые вопросы

## Общие вопросы

### Что такое SolidFlow Desktop?
SolidFlow Desktop - это бесплатное desktop-приложение для просмотра, анализа и подготовки 3D-моделей для 3D-печати и механической обработки.

### Какие форматы файлов поддерживаются?
В текущей версии (0.2.0) поддерживается только формат STL (бинарный и текстовый). Планируется добавление OBJ, STEP и PLY.

### Сколько стоит приложение?
Приложение абсолютно бесплатно и распространяется с открытым исходным кодом (лицензия MIT).

### На каких платформах работает?
SolidFlow Desktop работает на Windows, macOS и Linux.

### Нужен ли интернет для работы?
Нет, приложение работает полностью оффлайн после установки.

## Установка и запуск

### Как установить приложение?
1. Установите Python 3.10 или выше
2. Скачайте проект с GitHub
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите: `python -m solidflow`

### Не запускается приложение, что делать?
Проверьте:
* Установлен ли Python 3.10+
* Установлены ли все зависимости
* Активировано ли виртуальное окружение (если используете)

### Какие системные требования?
* Python 3.10 или выше
* 4 ГБ RAM (рекомендуется 8 ГБ)
* OpenGL совместимая видеокарта
* 100 МБ свободного места

## Работа с моделями

### Как открыть 3D-модель?
File → Open (Ctrl+O) → выберите STL файл

### Модель открылась, но ничего не видно
Попробуйте:
* Нажать Home для сброса камеры
* Проверить, что файл не пустой
* Масштабировать колесом мыши

### Как узнать размеры модели?
Размеры отображаются в панели информации справа после загрузки модели.

### Почему не показывается объем?
Объем вычисляется только для герметичных (watertight) моделей. Если модель имеет дырки, объем будет показан как 0.

### Как изменить размер модели?
В текущей версии нет функции масштабирования. Используйте другие CAD-программы для изменения размера перед импортом.

## Анализ и проверка

### Что значит "watertight"?
Watertight (герметичная) модель - это модель без дырок, полностью замкнутая. Это необходимо для 3D-печати.

### Что такое "non-manifold"?
Non-manifold геометрия - это некорректная топология, когда ребро принадлежит больше чем двум граням. Это может вызвать проблемы при печати.

### Что такое нормали?
Нормали определяют, какая сторона поверхности "лицевая". Неправильные нормали могут привести к некорректному отображению и проблемам при печати.

### Модель отображается "наизнанку"
Используйте: Инструменты → Исправить нормали

### Как проверить модель перед печатью?
1. Откройте модель
2. Проверьте статус в панели информации (должен быть зеленый)
3. Используйте Инструменты → Анализировать для детальной проверки
4. Убедитесь что модель watertight
5. Проверьте размеры

## Ремонт и обработка

### Что делает функция "Ремонт модели"?
Автоматически исправляет:
* Заполняет дырки
* Удаляет дублирующиеся вершины
* Удаляет вырожденные грани
* Очищает геометрию

### Безопасно ли использовать автоматический ремонт?
В большинстве случаев да, но рекомендуется сохранить оригинал перед ремонтом используя "Сохранить как".

### Ремонт не исправил проблемы
Некоторые сложные проблемы требуют ручного исправления в специализированных программах (Meshmixer, Blender, netfabb).

### Модель изменилась после ремонта
Ремонт может немного изменить геометрию при заполнении дырок. Всегда проверяйте результат визуально.

### Можно ли отменить изменения?
В текущей версии нет функции отмены (undo). Поэтому важно сохранять оригинал перед обработкой.

## Сохранение

### Как сохранить модель?
File → Save (Ctrl+S) - сохранить в текущий файл
File → Save As (Ctrl+Shift+S) - сохранить в новый файл

### В каком формате сохраняется?
Модели сохраняются в формате STL (бинарный).

### Можно ли сохранить в другой формат?
В текущей версии только STL. Поддержка других форматов планируется.

### Сохранение перезаписывает оригинал?
"Save" (Ctrl+S) - да, перезаписывает текущий файл
"Save As" (Ctrl+Shift+S) - нет, создает новый файл

## Производительность

### Приложение тормозит с большими моделями
Попробуйте:
* Закрыть другие приложения
* Упростить модель в другой программе
* Обновить драйверы видеокарты

### Сколько треугольников может обработать?
Обычно до 1-2 миллионов треугольников работает хорошо. Зависит от вашего компьютера.

### Ускорить работу приложения?
* Используйте режим каркаса для больших моделей
* Закройте ненужные приложения
* Добавьте RAM если возможно

## Ошибки

### "Не удалось загрузить файл"
Проверьте:
* Файл существует и доступен
* Формат файла - STL
* Файл не поврежден
* Есть права на чтение

### "Не удалось сохранить файл"
Проверьте:
* Есть права на запись в папку
* Достаточно места на диске
* Файл не открыт в другой программе

### Приложение зависло
* Дождитесь завершения операции (особенно для больших файлов)
* Если не отвечает долго - закройте и перезапустите
* Проверьте лог-файлы в папке проекта

## Возможности

### Можно ли печатать прямо из приложения?
Нет, приложение только подготавливает модели. Для печати используйте слайсер (Cura, PrusaSlicer и т.д.).

### Есть ли поддержка цвета/текстур?
Нет, в текущей версии только геометрия.

### Можно ли редактировать модель?
Нет, редактирование не поддерживается. Используйте CAD-программы для редактирования.

### Будет ли добавлена функция X?
Проверьте план развития в [docs/mvp-plan.md](mvp-plan.md) или создайте запрос на GitHub.

### Где посмотреть планы развития?
В файле [docs/features.md](features.md) описаны все планируемые функции.
