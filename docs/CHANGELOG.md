## [0.3.0] - 05.01.2025

### Добавлено - MVP Этап 5: Экспорт и финализация
* Полный экспорт STL (Save / Save As)
* Tooltips для всех действий в меню и toolbar
* Детальная обработка ошибок с подсказками
* Индикаторы прогресса (cursor + statusBar)
* Диалог подтверждения перед ремонтом
* Скрипт тестирования полного цикла (scripts/test_full_cycle.py)
* Документация по тестированию (docs/TESTING.md)
* Отслеживание изменений (флаг is_modified)

### Улучшено
* UX всех операций с файлами
* Сообщения об ошибках стали подробными и понятными
* Диалог "Сохранить как" предлагает имя с суффиксом _edited
* Автоматическое добавление расширения .stl
* Результаты ремонта показывают статус модели

### Документация
* docs/TESTING.md - полное руководство по тестированию
* docs/mvp-stage5-report.md - отчет о завершении этапа 5
* docs/USER_GUIDE.md - полное руководство пользователя  
* docs/FAQ.md - часто задаваемые вопросы

### Прогресс MVP
* ~95% завершено
* Готов полный цикл: импорт → анализ → ремонт → экспорт

## [0.2.0] - 2025-01-04

### Added - MVP Этап 4: Базовая обработка и анализ
* Модуль валидации mesh (MeshValidator)
  * Проверка watertight
  * Проверка manifold геометрии
  * Проверка нормалей
  * Обнаружение вырожденных граней
  * Поиск дублирующихся вершин
* Модуль статистики (MeshStatistics)
  * Геометрическая информация
  * Размеры модели
  * Вычисление объема
  * Вычисление площади поверхности
* Модуль обработки mesh (MeshProcessor)
  * Ремонт модели (заполнение дырок, удаление дефектов)
  * Исправление нормалей
  * Удаление дубликатов
  * Упрощение mesh
  * Сглаживание
* Модули импорта/экспорта (STLImporter/STLExporter)
* Меню "Инструменты":
  * Анализировать модель
  * Ремонт модели
  * Исправить нормали
* Автоматический анализ при загрузке модели
* Расширенная панель информации с валидацией
* Функциональность Save/Save As
* Визуальные индикаторы проблем (цветовая кодировка)
* Unit-тесты для новых модулей
