## [0.2.0] - 2025-01-04

### Added - MVP Этап 4: Базовая обработка и анализ
* Модуль валидации mesh (MeshValidator)
  * Проверка watertight
  * Проверка manifold геометрии
  * Проверка нормалей
  * Обнаружение вырожденных граней
  * Поиск дублирующихся вершин
* Модуль статистики (MeshStatistics)
  * Геометрическая информация
  * Размеры модели
  * Вычисление объема
  * Вычисление площади поверхности
* Модуль обработки mesh (MeshProcessor)
  * Ремонт модели (заполнение дырок, удаление дефектов)
  * Исправление нормалей
  * Удаление дубликатов
  * Упрощение mesh
  * Сглаживание
* Модули импорта/экспорта (STLImporter/STLExporter)
* Меню "Инструменты":
  * Анализировать модель
  * Ремонт модели
  * Исправить нормали
* Автоматический анализ при загрузке модели
* Расширенная панель информации с валидацией
* Функциональность Save/Save As
* Визуальные индикаторы проблем (цветовая кодировка)
* Unit-тесты для новых модулей
