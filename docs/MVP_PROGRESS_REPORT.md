## Проблемы и решения

### Решенные проблемы:
1. **Интеграция PyVista + Qt**
   * Решение: использование pyvistaqt.QtInteractor
   
2. **Структура проекта**
   * Решение: модульная архитектура с четким разделением

3. **Тестирование GUI**
   * Решение: pytest-qt для Qt компонентов

### Текущие ограничения:
* Только STL формат (запланированы OBJ, STEP)
* Одна модель за раз
* Базовая валидация
* Нет undo/redo
